# Tablero táctico: saleros vs limones

Proyecto web ligero y responsivo con formación base 1-4-2-3-1.

## Incluye
- Cancha estilo pizarra táctica
- 22 piezas arrastrables
- Salero y limón usando los PNG incluidos en `assets/`
- Número centrado sobre cada pieza
- Guardado automático en el navegador
- Botones para reiniciar e invertir lados

## Ejecutar
Abre `index.html` en tu navegador o súbelo a GitHub Pages.
